# Исследование финансирования стартапов


## 📊 Обзор проекта

Цель: Провести исследовательский анализ исторических данных о финансировании стартапов для выявления закономерностей и оценки перспектив выхода на рынок с покупкой и развитием компаний.

Автор: Земцова Ксения
Дата выполнения: 20.03.2026


## 📁 Данные

Проект использует два датасета:

Основной датасет (`cb_investments.csv`, 54 294 записи, 40 полей)
- `name` — название компании
- `category_list` — категории деятельности (через `|`)
- `market` — основной рынок компании
- `funding_total_usd` — общий объем привлеченных инвестиций
- `status` — текущий статус компании
- `country_code`, `state_code`, `region`, `city` — географические данные
- `funding_rounds` — число раундов финансирования
- `participants` — число участников в раундах
- `founded_at`, `first_funding_at`, `mid_funding_at`, `last_funding_at` — временные метки
- типы финансирования: `seed`, `venture`, `equity_crowdfunding`, `undisclosed`, `convertible_note`, `debt_financing`, `angel`, `grant`, `private_equity`, `post_ipo_equity`, `post_ipo_debt`, `secondary_market`, `product_crowdfunding`
- раунды финансирования: `round_A` — `round_H`

Дополнительный датасет (`cb_returns.csv`, 15 записей, 14 полей)
- `year` — год возврата средств
- те же типы финансирования с суммами возвратов в млн $


## 📈 Этапы анализа

1. Загрузка и предобработка данных
   - Оптимизация типов данных (float64 → float32)
   - Приведение названий столбцов к единому стилю
   - Преобразование `funding_total_usd` в числовой формат
   - Обработка дат (преобразование в datetime)
   - Заполнение пропусков в текстовых полях заглушками
   - Удаление строк без информации о финансировании

2. Инжиниринг признаков
Группы по срокам финансирования:
   - Единичное финансирование — один раунд
   - Срок финансирования до года — между первым и последним раундом ≤ 365 дней
   - Срок финансирования более года

Классификация сегментов рынка:
   - Массовые (>120 компаний) — 49 сегментов, 88.6% компаний
   - Средние (35-120 компаний) — 57 сегментов, 9.4% компаний
   - Нишевые (<35 компаний) — 289 сегментов, 2.0% компаний

3. Анализ выбросов
   - Выявление аномалий методом IQR по каждому сегменту
   - Удаление 5 244 компаний с аномальным финансированием (12.8%)
   - Проверка полноты данных за 2014 год
   - Фильтрация по годам с ≥50 раундами (удалено 0.2% компаний)

4. Анализ типов финансирования
   - Сравнение объемов и популярности 13 типов инвестиций
   - Анализ суммарных возвратов по типам

5. Динамический анализ
   - Динамика размера раунда по годам
   - Динамика количества раундов
   - Рост массовых сегментов в 2014 году
   - Нормированные значения возврата средств по типам
